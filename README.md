# JornadaMilhas - Aplicação Vue.js com TypeScript para Venda de Passagens Aéreas


## Tecnologias Utilizadas

* **Frontend:**
    * Vue.js
    * TypeScript
    * Vuex
    * BootStrap
    * CSS3
    * Vuelidate
    * Vue Router
    * Axios
    * Node.js
    * TypeScript
* **Arquitetura e Design:**
    * Clean Architecture
    * Repository Pattern
    * ResultValue
    * SOLID principles
    * Tsyringe (injeção de dependência)
* **Infraestrutura:**
    * Docker
    * Kubernetes
    * GitHub Actions
